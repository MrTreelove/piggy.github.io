# Piggy - 智能记账本应用

基于 React + Vite + Tailwind CSS 构建的现代化记账本应用 UI。

## 功能特性

- 🔐 **用户认证** - 登录、注册、密码管理，确保数据安全
- 📊 **仪表盘** - 总览财务状况，快速查看账户余额和收支统计
- 💰 **记账功能** - 支持收入、支出、转账三种类型，快速记录每一笔交易
- 📝 **记录管理** - 查看、搜索、筛选所有交易记录
- 📈 **统计分析** - 多维度数据统计，可视化图表展示
- 💳 **账户管理** - 管理多个账户，支持现金、银行卡、支付宝、微信等
- 🎯 **预算管理** - 设置预算，实时监控预算使用情况
- ⚙️ **设置中心** - 个性化设置，用户信息管理，数据导入导出
- 🔒 **数据隔离** - 每个用户仅能查看和管理自己的数据

## 技术栈

- **React 18** - UI 框架
- **Vite** - 构建工具
- **React Router** - 路由管理
- **Tailwind CSS** - 样式框架
- **Recharts** - 图表库
- **Lucide React** - 图标库

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

应用将在 `http://localhost:5173` 启动

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 页面说明

### 0. 认证页面

#### 登录页面 (Login)
- 用户名/邮箱登录
- 密码登录
- 记住登录状态
- 忘记密码链接

#### 注册页面 (Register)
- 用户注册（用户名、邮箱、密码）
- 密码强度验证
- 用户协议确认

#### 忘记密码页面 (ForgotPassword)
- 通过邮箱重置密码
- 发送重置链接

### 1. 仪表盘 (Dashboard)
- 总资产、本月收入、支出、结余概览
- 账户余额列表
- 最近交易记录

### 2. 记账页面 (AddTransaction)
- 支持收入、支出、转账三种类型
- 金额、分类、账户选择
- 日期、备注、附件上传

### 3. 记录列表 (TransactionList)
- 所有交易记录展示
- 搜索和筛选功能
- 编辑和删除操作

### 4. 统计分析 (Statistics)
- 支出分类占比（饼图）
- 月度收支对比（柱状图）
- 收支趋势（折线图）

### 5. 账户管理 (Accounts)
- 账户列表展示
- 添加、编辑、删除账户
- 账户余额查看

### 6. 预算管理 (Budget)
- 总预算设置和进度
- 分类预算管理
- 预算超支提醒

### 7. 设置页面 (Settings)
- **用户信息管理** - 修改用户名、邮箱
- **密码管理** - 修改密码功能
- 基础设置（货币、日期、主题）
- 安全设置（密码保护、数据加密）
- 通知设置
- 数据管理（导入、导出、清空）

## 设计特点

- 🎨 **现代化设计** - 简洁美观的界面，清晰的视觉层级
- 📱 **响应式布局** - 支持桌面端和移动端
- 🎯 **用户友好** - 直观的操作流程，快速完成记账
- 🎭 **流畅动画** - 平滑的过渡效果，提升用户体验

## 认证功能

### 用户认证
- 所有页面都需要登录后才能访问
- 未登录用户自动重定向到登录页面
- 登录状态保存在 localStorage（可选）
- 支持登出功能

### 数据隔离
- 每个用户只能查看和管理自己的数据
- 账户、交易记录、分类、预算等数据完全隔离
- 所有数据操作都关联到当前登录用户
